A data visualisation project based on data from Singapore Pool's Toto lottery results. Navigate using the buttons below to view different graphs with customisable date filters. If the graph has a legend, clicking an item in the legend will remove it from view and double-clicking a legend will isolate only that view. This is meant to be a fun personal project, note that lotteries are completely random and gambling is a risk. Please gamble responsibly. If you or someone you know is suffering from gambling addiction please contact the National Problem Gambling Helpline: 1800-6-668-668.

Currently a work in progress, the project is meant to be an outlet for me to apply my different skills and techniques I have learnt over the years. If you have any feedbacks or suggestions please connect with me on LinkedIn or through my email provided in the footer.
