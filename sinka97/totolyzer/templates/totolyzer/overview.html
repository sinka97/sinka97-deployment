A project based on data from Singapore Pool's ToTo lottery. Navigate using the buttons below to view different graphs with customisable date filters. This is meant to be a fun personal project, note that lotteries are completely random and gambling is a risk. Please gamble responsibly. If you or someone you know is suffering from gambling addiction please contact the National Problem Gambling Helpline: 1800-6-668-668.

Currently a work in progress, the project aims to find meaningful trends in a field that is apparently random. Will it be successful? I doubt it :( . Why do I still do it? I thought it will be fun. Did I have fun? Not really, it was actually quite tedious, but what matters is I learnt a lot :) !
