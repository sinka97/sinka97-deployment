Personal Projects

Totolyzer

A fun personal project, I built to practice my skills of data engineering and data analytics skills. The interactive dashboard allows users to study the trends and perform analysis through interactive graphs based on the Toto results data from Singapore pools. The seed data for the historical toto results were obtained from https://en.lottolyzer.com/ as a csv file and ingested into a postgres database via a python script. Subsequent data is scraped daily from Singapore Pools website itself to enhance the database. The dashboard is built using dash and plotly while the process of scheduled data ingestion is currently a work in progress with plans to integrate weather data in the future (Estimated to be completed 1st July 2023).
